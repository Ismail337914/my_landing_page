# WebZaBiznis (Next.js)

Migrisana landing stranica na Next.js (`app` router), uz zadržan isti sadržaj i modernizovan UI.

## Pokretanje

```bash
npm install
npm run dev
```

Otvori `http://localhost:3000`.

## Skripte

- `npm run dev` - Next.js development server
- `npm run build` - production build
- `npm run start` - pokretanje production builda
- `npm run lint` - lint provjera

## Struktura

- `app/layout.js` - root layout + metadata
- `app/page.jsx` - kompletan sadržaj landing stranice
- `app/globals.css` - globalni modernizovani stilovi, animacije i responsive pravila
